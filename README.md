1、采集了亚马逊文学图书书籍ID、名称、价格、评论数、URL
2、通过亚马逊图书名称，通过京东搜索，采集了相应的京东自营图书价格、评论数、URL，实现比价功能
3、通过书籍url+书籍价格构建信息指纹，采用Redis位数组并构建K个哈希函数的布隆过滤器进行书籍信息的去重，
   后期可采用爬取到具体的url存储Redis进行任务分发，并通过构建信息指纹来实现价格的更新
4、存储mysql数据亚马逊图书数据(24486)，相对应的京东数据(18857),
   具体数据查看文件 ./literaturebooks_distributed_data.sql
5、通过scrapy_redis持久化队列实现分布式爬取，./LiteratureBooks/spiders/MyRedis.py将信息push到队列，
   增大并发数、开启DNS缓存等，并部署到腾讯云ESC(Ubuntu)和本机(Windows10),进行分布式爬取

注:由于没有说明采集哪种价格(平装、电子书、精装),所以本程序控制了爬取深度,
   故采集了亚马逊图像视图的价格区间，所以数据量可能较小
